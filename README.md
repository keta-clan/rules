# keta-clan static site

>[VuePress](https://github.com/vuejs/vuepress) project. Generates the static site [https://keta-clan.github.io/](https://keta-clan.github.io/)

## When you writing code:
---
```
yarn docs:dev
```

## Generate static site:
---
```
yarn docs:build
```
